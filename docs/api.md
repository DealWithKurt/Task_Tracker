# Task Tracker API

Этот проект использует FastAPI и PostgreSQL для управления задачами на бекенде.

## Эндпоинты

### Задачи (Tasks)
- `GET /tasks` — Список задач с пагинацией, фильтрацией и поиском
- `GET /tasks/{id}` — Детальная информация о задаче
- `POST /tasks` — Создание новой задачи
- `PUT /tasks/{id}` — Обновление задачи
- `DELETE /tasks/{id}` — Удаление задачи (только admin)

### Комментарии (Comments)
- `GET /tasks/{id}/comments` — Получить комментарии к задаче
- `POST /tasks/{id}/comments` — Добавить комментарий к задаче
- `DELETE /comments/{id}` — Удалить комментарий

### Пользователи (Auth)
- `POST /auth/register` — Регистрация нового пользователя
- `POST /auth/login` — Вход, получение JWT-токена
- `GET /auth/me` — Информация о текущем пользователе

### Проверка здоровья (Health Check)
- `GET /health` — Статус API
- `GET /health/db` — Статус подключения к базе данных

## Формат ответа для списков
```json
{
  "items": [],
  "total": 100,
  "page": 1,
  "limit": 10,
  "pages": 10
}
```
